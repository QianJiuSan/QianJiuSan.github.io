Supervised Learning(自我监督学习)

Let’s start by talking about a few examples of supervised learning problems.Suppose we have a dataset giving the living areas and prices of 47 houses from Portland,Oregon:
让我们从几个监督学习的例子开始讲起。假如我们有47组关于居住面积和价格的Portland,Oregon的数据集：

We can plot this data:
我们用这些数据制图：

Given data like this,how can we learn to predict the prices of other houses in Portland,as a function of the size of their living areas?
数据如此，我们如何学会根据这些关于房屋的居住面积的函数去预测在Portland的房屋价格？
To establish notation for future use,we’ll use x[i] to denote the “input” variables (living area in this example),also called input features,and y[i] to denote the “output” or target variable that we are trying to predict (price).
确立将要运用的记号，我们将用x[i]代表输入数据（这里居住面积为例），所以称为输入特征量，并且y[i]代表输出或者称为我们试图预测的目标值（如价格）。
A pair (x[i],y[i]) is called a training example, and the dataset that we’ll be using to learn——a list of m training examples {(x[i],y[i]);i = 1,…,m}——is called a training set.
一对(x[i],y[i])称为一组训练例，我们将用去进行机器学习的数据称为训练组，是一组有m个数据对的列表。
Note that the superscript “i” in the notation is simply an index into the training set,and has nothing to do with exponentiation.
写在边上的i只是一个关于训练集的简单的标志，并不是做求幂运算。
We will also use X denote the space of input values,and Y the space of output values.In this example,X=Y=R.
我们将用大X来表示输入值的数集，用大Y来表示输出值的数集。在这个例子里，X与Y属于实数集。
To describe the supervised learning problem slightly more formally,our goal is,given a training set ,to learn a function h:X->Y so that h(x) is a “good” predictor for the corresponding value of y.
为了更直接更正式地描述监督学习的问题，我们的目的是，给予一组训练集，去学习模拟得到一个由X映射到Y的函数h，使h(x)能够很好的预测到相应的y值。
For historical reasons,this function h is called a hypothesis.Seen pictorially,the process is therefore like this:
于经验上讲，函数h(x)称为一个假说。用图表示这个过程，是这样的：

When the target variable that we’re trying to predict is continuous,such as in our housing example,we call the learning problem a regression problem.When y can take on only a samll number of discrete values (such as if,given the living area,we wanted to predict if a dwelling is a house or an apartment,say),we call it a classification problem.
当我们试图去预测的目标值使是连续的，就像我们房屋的例子，我们把这类问题称为回归问题。当y只有一小部分的离散值时(这样说，如果，我们拥有居住面积，我们想去预测这件屋子是平房还是公寓)，我们称这是分级问题。
